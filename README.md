## ⚙️ Cookbook Developer Setup

**Configure Cookbook Template**

```
# fork/clone repo
git clone https://github.com/nogibjj/candle-cookbook

# install mdbook
make install

# serve/preview cookbook at localhost:8000
make serve

# build cookbook
make cookbook
```
